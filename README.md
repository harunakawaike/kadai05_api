# ①課題名
推し活聖地巡礼マップ

## ②課題内容（どんな作品か）
- YouTubeやテレビ番組の情報をもとに、推しが訪れた場所を地図上に表示する聖地巡礼アプリ
- テーマカラーを切り替えられるようにして、推しのメンバーカラーに合わせて画面の雰囲気を変えられる。
- 検索した動画から場所候補を抽出し、巡礼ルートの順番も見られる。

## ⑤工夫した点・こだわった点
- 推し活らしさを出すために、テーマカラーを複数用意して画面全体の印象を変えられるようにした。
- サイドバーでテーマを切り替えられるようにして、操作しやすいUIを意識。
